# Contributing

We welcome contributions to the Voice Agent Starter. Here’s how to get involved.

## Reporting issues

- Open a [GitHub Issue](https://github.com/murf-ai/murf-livekit-starter/issues) with a clear description of the bug or feature request.
- Include steps to reproduce (for bugs), your environment (OS, Node/Python versions), and any relevant logs or screenshots.

## Submitting changes

1. Fork the repo and create a branch from `main` (e.g. `fix/typo-readme` or `feat/add-example-prompt`).
2. Make your changes. Keep the scope small and focused.
3. Run any existing tests (e.g. `cd backend && uv run pytest`).
4. Open a Pull Request with a short description of what changed and why.
5. We’ll review and may ask for tweaks before merging.

## Questions

For questions or discussion, join the [Murf Discord](https://discord.gg/FbKAy96Sz7).

Thanks for contributing.
